
API - Livros
1 - Objetivos: Criar uma API que disponibiliza a consulta, criação, edição e exclusão de Livros
2 - URL base - localhost
3 - EndPoints -

    - localhost/livros (GET)
    - localhost/livros/id (GET)
    - localhost/livro/id (PUT)
    - localhost/livro/id (DELETE)

4 - Utiliza:

    - Flask
    - Python
    - Postman
